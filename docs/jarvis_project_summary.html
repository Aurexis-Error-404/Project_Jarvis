Project JARVIS — Proactive Developer Intelligence Layer

Generated: 2026-04-09T18:32:47.260829Z

Executive Summary

**JARVIS** is a desktop application for Windows that delivers **proactive developer intelligence** using a **zero-search paradigm**. Instead of waiting for developers to ask questions, JARVIS detects what you're working on and surfaces relevant insights automatically.

- **Core Philosophy**: Zero-Search — context surfaces before you ask
- **Build Timeline**: 48-hour sprint with defined gates and feature freeze at Hour 36
- **Team Size**: 5 members across AI, Backend, Frontend, Integration, and Research roles

Technology Stack

| Layer | Technology |

| Backend | Python FastAPI + WebSocket server |
| AI Cloud | Claude API (Sonnet-4, Haiku-4) |
| AI Local | Ollama + CodeLlama (RTX 4090) |

AI Architecture Decisions

| Task | Chosen Model | Rejected | Reason |
|---|---|---|---|
| Proactive Gate | Ollama / CodeLlama | Claude API | Gate runs 50+ times/hour — must be free and local |
| Error Diagnosis | gemini-2.5-flash | llama-3.3-70b-versatile | User-facing quality needs deep reasoning |
| Summaries & Commits | llama-3.3-70b-versatile | gemini-2.5-flash | Groq matches quality at lower latency/cost |
| Project Memory | jarvis.json schema | Free-text notes | Structured fields prevent hallucination |
| Secure Mode | CodeLlama via Ollama | Claude API | Zero bytes leave machine in secure mode |
