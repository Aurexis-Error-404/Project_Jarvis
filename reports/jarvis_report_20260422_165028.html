Executive Summary

KrishiCFO is a season-wise crop intelligence dashboard built to help users compare commodity performance across marketing seasons (2022-23 through 2025-26). The project focuses on MSP vs seasonal price analysis and Kharif/Rabi arrival visibility — not mandi-level forecasting or daily time-series predictions.

Current Status: Data layer complete with normalization scripts and merged outputs. Frontend scaffolding in progress with core components implemented.

Dataset Overview

The crop_data/ folder contains four checked-in season reports:

  • 2022-23
  • 2023-24
  • 2024-25
  • 2025-26

Each report includes: Commodity Group, Commodity, MSP, Kharif Price/Arrival, Rabi Price/Arrival.

Generated Outputs:

  • season_report_summary.csv — normalized merged dataset
  • season_report_summary.json — JSON version for frontend consumption

Data Rules:

  • Source commodity names preserved exactly
  • - converted to null
  • Numbers parsed to numeric values
  • Raw files remain untouched

Current Limitations (By Design)

This dataset version does not support:

  • Mandi filters or location-based analysis
  • Daily time-series forecasting
  • Transport/profit logic by location
  • Week-level sell recommendations

Future Scope: These features require mandi-level daily data, which is not yet integrated.

Key Decisions Locked
