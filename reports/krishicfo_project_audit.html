KrishiCFO MVP: Season-wise Crop Intelligence Dashboard

Project Overview

KrishiCFO is a season-wise crop intelligence dashboard built on real agricultural data from 2022-23 through 2025-26. The project focuses on commodity comparison across seasons, MSP vs seasonal price analysis, and Kharif/Rabi arrival visibility — deliberately avoiding mandi-level forecasting until richer data is available.

Technology Stack

  • Frontend: Next.js 14 + React 18 + TypeScript
  • Backend: FastAPI + Python (planned)
  • Data: Season-wise CSV reports (4 seasons, normalized to JSON)
  • Deployment: Local-first with future API wrapper support

Architecture Details

Data Layer (crop_data/)

  • 4 raw season reports (2022-23 through 2025-26)
  • Normalized fields: season_year, commodity_group, commodity, msp, kharif_price, kharif_arrival_tonnes, rabi_price, rabi_arrival_tonnes
  • Missing values (-) converted to null in JSON

Frontend Components

  • CommodityGroupSelector — filter by crop category
  • CommoditySelector — filter by specific commodity
  • SeasonPriceChart — multi-season price movement
  • SeasonArrivalChart — Kharif/Rabi arrival comparison
  • MSPComparisonCard — price vs MSP analysis
  • CommoditySummaryTable — tabular overview
  • InsightCards — automated insights
