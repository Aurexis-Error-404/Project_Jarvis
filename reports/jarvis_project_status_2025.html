Executive Summary

JARVIS is a proactive developer intelligence layer built on Electron, React, and FastAPI, implementing a zero-search paradigm for ambient coding assistance. The system uses local LLMs (Ollama/CodeLlama) for the proactive gate and cloud models (Gemini, Groq) for user-facing quality tasks.
